武当攻略

发表于2025-02-06|更新于2025-03-21
|总字数:5.5k|阅读时长:16分钟

武当注意事项

武当是终老门派，可以从桃花转来武当或者硬怼太极篇。否则打宗师塔和情怀岛的武当宗师就是送死

武当宗师解密

本站武当宗师是yun yinyun中达到要求解开，所以他是不定时的。武当从你参悟满属性开始就要吃珍珠，我一般吃10年的珍珠一辈子都要续珍珠，除非你已经解开了武当宗师。武当宗师是武当派最强quest没有之一

武当绝技

爬塔最快是剑指术，剑指术要求jifa sword taiji-jian;jifa parry taiji-quan。

通吃所有绝技是太极术，当你解了武当宗师且有太极术就成为武当究极形态。

武当小秘密

set 太剑神功

set 太极剑 挑

set taiji 挤|采|采|采|采|采|挤|采|采|采|采|采
