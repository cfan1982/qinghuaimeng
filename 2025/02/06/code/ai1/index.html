用AI看懂Lpc和Lua代码(通译灵码篇)

发表于2025-02-06|更新于2025-02-07
|总字数:324|阅读时长:1分钟

VSCode安装配置

1. 下载安装VSCode

首先需要下载VSCode编辑器：

  • 官网下载地址：Visual Studio Code - Code Editing.

2. 设置中文界面

安装中文语言包：

  1. 打开VSCode扩展商店
  2. 搜索并安装 Chinese (Simplified) 插件
  3. 重启VSCode后界面将变为中文

3. 安装通义灵码

  1. 在扩展商店中搜索 tongyilingma
  2. 安装通义灵码插件
  3. 按提示注册阿里账号并登录


安装完成后：

  • VSCode左侧边栏会出现通义灵码图标
  • 右键菜单中会增加通义灵码相关选项

使用通义灵码

生成代码注释

  1. 选择需要注释的代码片段
  2. 右键选择”通义灵码-生成注释”

查看生成的注释


如果生成的注释不够清晰，你还可以：

  • 继续向AI提问，请求更详细的解释
  • 尝试不同的提示方式来获取更好的注释效果

使用通义灵码的注意事项

  1. 通义灵码需要联网使用，请确保网络连接正常
  2. 通义灵码的注释功能依赖于AI模型的理解和解释能力，可能存在一定的误差
  3. 通义灵码的注释功能可能会受到网络环境的影响，请确保网络连接稳定
