用AI看懂Lpc和Lua代码(deepseek篇)

发表于2025-06-02|更新于2025-06-02
|总字数:1.3k|阅读时长:3分钟

【1】引言

最近deepseek很火，我也尝试了一下使用deepseek，确实效果显著。

一般来说，如果我懒得自己从头到尾写代码，往往是复制别人的成型代码，然后拿过来修修改改。还是要耗费一些时间，也必须对代码有所认识。

但实际工作面对的需求往往千奇百怪，要想高效满足所有需求，不得不想办法学习更过知识来满足工需求。但无论如何，学习总需要时间，也要切好找得到相应的学习资源。

AI在一定程度上缓解率为我们的学习焦虑，那进一步，如果它们直接输出代码，岂不是更好。现在已经有多种可以直接输出代码的AI ，但我们使用中也会遇到个各各样奇怪或者说令人哭笑不得的问题，比如满足了新要求却丢了旧要求，比如面对有些bug永远不会承认。

其实更令人难受的，有些时候你需要从AI的回答里复制代码自己去运行，然而当需求较多时，代码突然输出不全，并且提示对话字数已经满额。

所以，如果代码可以直接在编译器中输出，并且直接修正，下过可能会好很多。

deepseek恰好就能满足这样的需求。众所周知还有其他的工具也能满足这些需求，请大家按需使用。

【2】使用教程

第一步：

首先要有VScode，如果没有就安装一个。具体教程好多兄弟们的博客都讲了，此处不再赘述。

然后打开VScode，在扩展里面搜索Cline，选择第一个，记得确认它也是下载量最大的那一个：

图1

然后安装这个扩展，非常快。我这里的828ms表示我已经安装过了，大家点击安装即可。

第二步：

Cline安装完成之后，会在VScode的左侧工具栏看到一个类似机器人的图标，点击一下它：

图2

第三步：

点完机器人图标之后就来到一个配置界面，记住点击齿轮进行配置，首先把API Provider选择为“OpenAI Compatible”：

图3

第四步：

点击齿轮settings会进入图3这个界面，界面标出了1234一共四个框，这是需要填写除API Provider的配置内容。

如何填写配置内容，就要进入deepseek官网获得相关信息，官网链接为：

DeepSeek

进入官网，会看到下述界面：

图4

第五步：

在官网首页点击“接入API”，首次点击需要使用手机注册。目前可以放心注册，输入手机号，获取验证码，然后就可以进入登陆后的界面：

图5

图5所示的页面是我的账户登录成功后的界面，可以看到网站一共赠送了10元的额度，我已经使用0.26元，大家初次打开应该都是10元满额。

第六步：

点击登陆成功页面的API keys，选择创建API：

图6

第七步：

在创建API页面，名称可以随便写，然后点击创建，会生成一个key，点击复制保存这个key，因为之后它就不会再完整显示了：

图7

然后在这个登录成功界面会看到你创建的所有API：

图8

第八步：

然后再回到页面左侧，会有一个“接口文档”，点击它：

图9

然后我们就会进入接口文档的说明界面，在这个界面，会找到图3所需要的部分信息：

图10

在这里，图3需要的Base URL和Model ID就找到了。然后API Key在第七步你已经复制了，把它填进去就行。

第九步：

填完信息记得点击Done:

图11

第十步：

然后输入你的想法，开始你愉快的coding。

图12

【3】总结

提供了VScode配置deepseek为API实现代码自动生成的技巧。

【4】zmuder注意事项

提醒一下zmuder，本文介绍方法经过本人实践可正常使用，但deepseek API是需要付费的，如果你不想付费那么请自己查找可用的deepseek API或者使用网页版
或者本地架设deepseek模型，但是本地模型对电脑要求高且需要一定的技术水平，如果你有兴趣可以自己查找资料进行尝试。deepseek第三方免费chat或者收费的API，
暂时我只收集到火山引擎、腾讯元宝、硅基流动。

【5】已知免费deepseek api

阿里推出的通译灵码经过更新后新增了deepseek R1，可以免费使用，但是使用体验不如qwq-plus，可以自行体会使用哪个模型。
